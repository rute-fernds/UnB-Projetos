#### Projeto Final: Relógio Digital com Cronômetro e Timer 🕑

O projeto final da disciplina de LCL consistiu no desenvolvimento de um relógio digital com funcionalidades de cronômetro e timer. Para isso, utilizamos o software Quartus II para a implementação do código em Verilog e a construção dos circuitos digitais. Além disso, usamos também a placa FPGA DE2, da Altera, para a síntese, testes e verificação do hardware desenvolvido.

#### 💡 Minha contribuição
Participei de todas as etapas do desenvolvimento do projeto, destacando:
- Modelagem da lógica dos circuitos;
- Construção dos circuitos digitais no Quartus II;
- Desenvolvimento e implementação dos contadores em Verilog;
- Integração e testes no Quartus II;
- Verificação do funcionamento na FPGA.

Embora o projeto tenha sido entregue funcionalmente, algumas funcionalidades planejadas não puderam ser concluídas devido à complexidade do sistema e ao tempo disponível para o desenvolvimento.

➡️ Para mais detalhes sobre a documentação e o código do projeto, acesse [LCL](https://github.com/rute-fernds/UnB-Projetos/tree/main/2024/LCL%20-%202024.1/Projeto%20Final%20D8)

## 📅 2025.2 - 5° Semestre

### 🎲 Banco de Dados (BD)

Para a disciplina de Banco de Dados, desenvolvi um sistema de administração de condomínios com CRUD completo. O objetivo desse projeto era desenvolver uma aplicação capaz de modernizar e otimizar os processos administrativos realizados por gestores condominiais, aplicando os conceitos básicos de Banco de Dados juntamente com o MySQL, SGBD utilizado no projeto.

#### Integrantes  👥
- [Mariana Siminion](https://github.com/marianassimion)
- [Rute Fernandes](https://github.com/rute-fernds)

Durante o desenvolvimento, fiquei responsável principalmente pela criação das telas utilizando Streamlit, pela elaboração dos diagramas MR e MER, por parte da estruturação do banco de dados e por grande parte do desenvolvimento e implementação do CRUD do sistema. Além disso, criei e implementei a view do banco de dados, um dos pré-requisitos do projeto.

➡️ Para mais detalhes sobre a documentação e o código do projeto, acesse [Gerenciador-Condominios](https://github.com/rute-fernds/Gerenciador-Condominios).
